## RESEARCH AREA 1: Dropship vs Wholesale Models Across Platforms

### Sources Researched:
- TikTok Shop Partner Center policies & seller guidelines
- Amazon Seller Central (FBA vs FBM vs Dropship)
- Shopify Partner ecosystem (DSers, Spocket, Printful, Syncee, Zendrop)
- Pinterest Commerce & Shopping Pins documentation
- Faire, Tundra, CREOATE wholesale platforms
- 3PL comparisons (ShipBob, Deliverr/Shopify Fulfillment Network)

### Key Findings:

#### Platform-Specific Fulfillment Rules:
| Platform | Dropship Allowed? | Wholesale/FBA? | Affiliate? | Key Constraint |
|----------|-------------------|----------------|------------|----------------|
| TikTok Shop | Yes (fast ship only) | Yes | Yes (built-in) | Must ship within 2-3 days; long-ship = suspension |
| Amazon | Yes (strict rules) | Yes (FBA preferred) | Yes (Associates) | Must be seller of record; no 3rd-party branding |
| Shopify | Yes (no restrictions) | Yes | Yes (Collabs) | Full flexibility, any model works |
| Pinterest | Via linked store | Via linked store | Yes (affiliate pins) | Traffic driver, not direct seller |

#### Economic Comparison:
| Model | Upfront Cost | Typical Margin | Risk | Speed to Market |
|-------|-------------|----------------|------|-----------------|
| Dropship | $0 | 10-30% | Low | Days |
| Wholesale/Bulk | $500-5,000 | 30-50% | Medium | Weeks |
| Private Label | $2,000-10,000+ | 40-70% | High | Months |
| Print-on-Demand | $0 | 15-35% | Low | Days |
| Affiliate | $0 | 1-45% commission | Zero | Immediate |

#### YOUSELL Product-to-Model Recommendation Logic:
- Physical product <$30 + fast shipping available → **Dropship** (US-based fulfillment)
- Physical product $30-100 + high demand → **Wholesale** (Faire/Alibaba + FBA)
- Physical product $100+ → **Wholesale/Private Label** only
- Digital product (any) → **Digital delivery** (create/license + sell on Gumroad/Etsy/Shopify)
- AI/Software → **Affiliate** (link + promote via content)
- Physical high-volume → **Wholesale** option should always be presented alongside dropship

| Kalodata | TikTok focus | 15-minute | Yes (rankings) |

#### Recommended Fallback Architecture for YOUSELL:

**Tier 1: Nightly Batch (Core Intelligence)** — 2 AM UTC daily
- Rescore all active products (free, compute only)
- Update trend stages and lifecycle
- Generate recommendation caches per client
- Compute dashboard widgets ("Today's Hot", "Rising Stars")
- Cost: $5-20/run

**Tier 2: Periodic Scrape (Fresh Data)** — Every 6-48 hours
- HOT products: Refresh every 6 hours
- WARM products: Refresh every 24 hours
- WATCH products: Refresh every 48 hours
- COLD products: Refresh weekly
- Cost: $10-50/run

**Tier 3: Weekly Deep Scan (Discovery)** — Sunday 3 AM
- Full product discovery across all 7 platforms
- New influencer/supplier discovery
- Cross-platform validation
- Cost: $30-100/run

**Tier 4: On-Demand (User-Triggered)** — Button click
- Live scrape for specific query
- Shows "Updating..." with progress
- Serves stale data immediately, refreshes in background
- Cost: $0.50-5/request

**Tier 5: Admin Manual Import** — CSV upload
- Parse + map + merge + score
- Free (data already purchased externally)

#### Dashboard Recommendation Widgets:
1. **"Today's Hot Products"** — Top 10 by final_score, daily refresh
2. **"Rising Stars"** — Score increase >10% in 7 days
3. **"Best for You"** — Personalized by client's platforms + niche
4. **"New Opportunities"** — Discovered in last 48h, score >60
5. **"Seasonal Picks"** — Pre-computed seasonal trends
6. **"Category Leaders"** — Top product per category

#### Smart Filter System:
- **Essential**: Platform, Score Tier, Price Range, Category, Trend Stage, Date, Sort
- **Advanced**: Fulfillment Model, Marketing Channel, Margin, Competition, Supplier Available, Influencer Matches, Cross-Platform
- **Presets**: "Quick Wins", "High Profit", "Trending Now", "Low Risk", "Affiliate Ready", "TikTok Viral"

| Source | Public API? | Data Export? | Pricing | Best Data |
|--------|-----------|-------------|---------|-----------|

| SerpAPI | Yes | N/A | $50-250/mo | Google Shopping, SERP data |
| Jungle Scout Cobalt | Yes (enterprise) | N/A | $500+/mo | Amazon deep intelligence |

#### Keepa API Integration:
- Endpoints: `/product`, `/search`, `/bestsellers`, `/deals`, `/category`
- Token system: 50 tokens/min regeneration
- Key data: Historical prices, BSR, Buy Box, offers, deals, review trends
- Best for: Enriching Amazon products with historical signals

#### CSV Import Strategy (FastMoss + Kalodata + Keepa):
1. Admin exports data from external tool
2. Uploads CSV to YOUSELL admin dashboard
3. System auto-detects source via column headers
4. Pre-built column mappings applied
5. Products matched by external_id/URL/title
6. New data merged into existing records
7. Scores recalculated with enriched data

#### Multi-Source Data Fusion Engine:
- Source reliability weights: Keepa API (0.95) > Admin import (0.85) > Apify scraping (0.70-0.75) > Google Trends (0.60) > Estimates (0.50)
- Freshness decay: <1hr (1.0), <6hr (0.95), <24hr (0.85), <72hr (0.70), <7d (0.50), <30d (0.30)
- Fusion rules: Price = latest timestamp; Sales = max credible source; BSR = Keepa only; Trends = weighted average; Scores = recalculate from fused data

#### New Database Tables Needed:
1. `import_history` — Track all admin uploads (file, rows, status)
2. `data_source_log` — Track enrichment sources per product
3. `product_score_snapshots` — Historical scores for trend detection
4. `column_mapping_templates` — Saved mappings for each data source
5. `client_preferences` — Preferred platforms, categories, price range
6. `recommendation_cache` — Pre-computed recommendations per client
